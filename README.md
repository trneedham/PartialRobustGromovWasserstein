# PartialRobustGromovWasserstein

Code for our paper "Metric Properties of Partial and Robust Gromov-Wasserstein Distances", Jannatul Chhoa, Michael Ivanitskiy, Fushuai Jiang, Shiying Li, Daniel McBride, Tom Needham, Kaiying O'Hare. The paper is available as an arXiv preprint: https://arxiv.org/abs/2411.02198.

The paper has a theoretical focus, but includes some simple numerical results. The code and notebooks in this repository reproduce those results.
